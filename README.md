> Yet another DiamondFire quality-of-life mod designed to make coding, moderation, and support sessions less painful.
> Also designed to be a less shitty rewrite of CodeClient [Latest Release](https://github.com/firef1a/codeutils/releases)

## Features
> Currently only contains a handful of features for DiamondFire staff and features with niche use-cases.
> Hopefully this will eventually expand include features from other popular DF mods like CodeClient.
> - **General**
    >   - Adds a quick join button to title screen.
>   - Simplifies staff chat tags.
>   - Adds a separate chat HUD for player messages, staff chat, and more.
>   - Auto-tip boosters
>   - a fuck ton of command aliases
      >     - /queue - /support queue
>     - /mv - /mod vanish
>     - /av - /admin vanish
>     - /nbt - /item nbt
>     - /ls - /lagslayer
>     - /debug - /item get debug_stick
>     - /cs <id> - /support code-spectate <id>
>     - /vars <pattern> - /p vars <pattern>
>     - more that arent listed here
> - **Developer**
    >   - Show code chest contents without opening them.
>   - Show full function and process names on code signs.
>   - Colors code signs.
>   - Shows custom DiamondFire item tags in item lore.
>   - Custom CPU display bar.
> - **Support**
    >   - Show support queue, current sessions, and support questions in the HUD.
>   - Automatically run /support queue when joining the server.
> - **Moderation**
    >   - Outline players of interest to make them easier to track for moderation purposes (/track <player>).
          >     - Show tracked players hit range and other statistics / abilities like if they are able to fly.
>   - Clicking on reports to automatically vanish and teleport to them.
>   - Remove spectator acceleration for more precise vanish movement.
> - & more cus ill prob forget to update this :3
