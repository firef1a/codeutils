## modmod
> DiamondFire quality of life mod for support and moderation staff. This is designed for DF staff, so you won't get much utility out of it if you aren't a staff member. Staff features will automatically be disabled if you don't have the required permissions or if you aren't on DF.<br><br>
> **NOTE TO DF MODERATORS: THIS MOD MAKES MISTAKES, DO NOT SOLELY RELY ON IT**<br>
> [How to use this mod](https://github.com/miawara/modmod/wiki)
